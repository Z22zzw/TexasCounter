# TexasCounter Project Context

## Project Overview
- **Name**: TexasCounter (WechatDeepSeekDemo1)
- **Type**: WeChat Mini-App + Node.js Backend
- **Purpose**: 线下德州扑克朋友局记分工具

## Tech Stack
- Frontend: UniApp + Vue3 (WeChat Mini Program)
- Backend: Node.js + TypeScript + Fastify
- Database: SQLite (better-sqlite3 + Drizzle ORM)
- Auth: jsonwebtoken (JWT)
- Money: decimal.js
- Test: Vitest + Supertest

## Key Architecture Decisions
- Ledger-based scoring (immutable entries)
- Owner-only settlement
- Integer betting input, 2-decimal display
- JWT auth with Bearer token
